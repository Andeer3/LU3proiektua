# Ilea-Apindegia

Ile-apaindegi baten eguneroko kudeaketa errazateko garatutako aplikazio bat da.
Bezeroak, hitzorduak eta zerbitzuak modu errazean administratzeko aukera ematen du.

programa hau C# erabiliz garatu da, ile-apaindegirako behar oinarrizikoak asetzeko.
Aplikazioak bezeroen informazioa gordetzea, hitzorduak kudeatzea eta eskaintzen diren
zerbitzuak kontrolatzea ahalbidetzen du.

## Ezaugarriak
- Bezeroen kudeaketa (gehitu, ezabatu, etab)
- Hitzorduen erregistroa
- Zerbiotzuen kudeaketa
- Sinplea eta erabiltzeko erraza

## Baldintzak
- Windows sistema eragilea izatea
- .NET Framework instalatuta izatea
- Visual Studio izate (gomendatua), beste bat erabil daiteke.

# Erabilera
Menu nagusitik aukeratu:
- Bezeroak kudeatu
- Hitzorduak sortu
- Datak aldatu
- Zerbitzuak ikusi edo aldatu
Nahi duzun aukera aukeratu behar duzu.

# Erabilitako teknologia
- C#
- .NET
- Visual Studio
- Windows 11 sistema eragilea
